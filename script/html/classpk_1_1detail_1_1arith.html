상세한 설명

value 타입. bool, int, float 등 기본 타입을 감싸는 클래스다.

arith value;
// 반드시 타입이 같아야 한다.
int i = 1;
value = i;
i = int(value);
float f = 1.f;
value = f;
f = float(value);
// 이렇게 사용하면 안됨!
value = i;
i = int8(value);
i = float(value);
// 이건 OK!
value = i;
f = int(value);

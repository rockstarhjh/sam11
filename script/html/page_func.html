설명

  • 특정 함수를 재정의하거나 후킹을 목적으로 합니다.
  • 하나의 함수만을 설정할 수 있습니다만 pk::get_func 함수를 통해 이전 핸들러를 얻어 체인을 만들 수도 있습니다. 체인을 만들때는 재귀가 일어나지 않도록 주의하시기 바랍니다.
  • pk::get_func 함수는 해당 번호에 함수가 설정되어 있지 않다면 원본 함수가 반환됩니다.
